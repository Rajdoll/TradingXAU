# Design: XAU Trading Assistant — Phase 1, Task 1.2 — State & Config

**Date:** 2026-03-30
**Status:** Approved
**PRD Reference:** XAU_Trading_Assistant_PRD.docx v1.0.0 §3.3, §4.7, §6.2, §9

---

## Decisions

| Decision | Choice | Rationale |
|---|---|---|
| State type | TypedDict | PRD-exact; LangGraph native; zero overhead |
| Config loading | `load_dotenv()` at import time | Self-contained; no boilerplate in callers |
| File split | state = TypedDicts only; config = constants + env | Clean boundary; matches PRD |

---

## `core/state.py`

Two TypedDicts. No logic, no imports beyond `typing`.

### PatternResult

| Field | Type | Description |
|---|---|---|
| `name` | `str` | e.g. `'Double Bottom'` |
| `win_rate` | `float` | e.g. `0.93` |
| `direction` | `str` | `'BUY'` or `'SELL'` |
| `formation_pct` | `float` | 0.0–1.0, pattern completeness |
| `entry_zone_low` | `float` | Lower bound of entry zone |
| `entry_zone_high` | `float` | Upper bound of entry zone |
| `invalidation_level` | `float` | Price that invalidates the pattern |
| `tier` | `int` | 1 (>80% WR), 2 (65–80%), 3 (<65%) |
| `skip` | `bool` | `True` for Descending Triangle only |

### XAUState (35 fields across 7 layers)

**Data Layer (7):** `ohlcv_h4`, `ohlcv_m15`, `fetch_timestamp`, `current_price`, `session`, `news_risk`, `news_events`

**Pattern Layer (3):** `patterns_found`, `active_pattern` (`Optional[PatternResult]`), `no_pattern`

**Confirmation Layer (6):** `rsi_value`, `rsi_signal`, `candlestick_confirm`, `candlestick_name`, `sr_context`, `tech_score`

**Timeframe Layer (6):** `h4_bias`, `h4_last_hh`, `h4_last_hl`, `m15_aligned`, `m15_entry_type`, `tf_verdict`

**Macro Layer (4):** `dxy_trend`, `tips_yield_direction`, `macro_bias`, `macro_confidence`

**Debate Layer (5):** `bull_arguments`, `bear_arguments`, `debate_score`, `debate_verdict`, `debate_key_reason`

**Risk Layer (11):** `account_balance`, `risk_pct`, `risk_usd`, `lot_size`, `sl_pips`, `sl_price`, `tp1_price`, `tp2_price`, `risk_reward`, `risk_approved`, `risk_reject_reason`

**Output Layer (5):** `confidence`, `signal_id`, `signal_summary`, `journal_written`, `pipeline_duration_s`

Note: `patterns_found` uses plain `list` (not `list[PatternResult]`) to avoid forward-reference issues at TypedDict definition time.
