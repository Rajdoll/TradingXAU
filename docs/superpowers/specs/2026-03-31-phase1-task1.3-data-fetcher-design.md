## Decisions

| Decision | Choice | Rationale |
|---|---|---|
| Structure | Private helpers + public node | Testable without mocking entire function |
| MultiIndex columns | `droplevel(1, axis=1)` after download | yfinance ≥0.2.50 returns MultiIndex for single ticker |
| Empty data | Raise `RuntimeError` | Fail fast — no silent bad state in pipeline |
| news_risk/news_events | Stub `False` / `[]` | Phase 2 — `tools/news_calendar.py` not yet implemented |
| State update pattern | `{**state, ...}` spread | LangGraph convention |

## Module: `agents/data_fetcher.py`

### Public interface

```python
def fetch_data_node(state: XAUState) -> XAUState
```

Called by LangGraph pipeline. Populates 7 state fields. Raises `RuntimeError` on fetch failure.

### Private helpers

```python
def _fetch_h4() -> pd.DataFrame
def _fetch_m15() -> pd.DataFrame
def _detect_session() -> str
```

---

## Implementation

### `_fetch_h4() -> pd.DataFrame`

1. `yf.download(TICKER, period="60d", interval="1h", progress=False)`
2. Raise `RuntimeError` if result is empty
3. Flatten MultiIndex: `df.columns = df.columns.droplevel(1)` if `isinstance(df.columns, pd.MultiIndex)`
4. Resample 1h → 4h: `.resample("4h").agg({"Open":"first","High":"max","Low":"min","Close":"last","Volume":"sum"})`
5. `.dropna().tail(H4_BARS)` — H4_BARS = 200
6. Raise `RuntimeError` if result is empty after resample

### `_fetch_m15() -> pd.DataFrame`

1. `yf.download(TICKER, period="5d", interval="15m", progress=False)`
2. Raise `RuntimeError` if result is empty
3. Flatten MultiIndex (same as H4)
4. `.dropna().tail(M15_BARS)` — M15_BARS = 200

### `_detect_session() -> str`

UTC-hour bucketing (no DST handling — session is a context label, not a trade gate):

| UTC hours | Session |
|---|---|
| 09:00–13:59 | `"LONDON"` |
| 14:00–23:59 | `"NEW_YORK"` |
| 00:00–08:59 | `"ASIAN"` |

### `fetch_data_node(state: XAUState) -> XAUState`

```python
{
    **state,
    "ohlcv_h4":        h4.to_dict(),
    "ohlcv_m15":       m15.to_dict(),
    "current_price":   float(h4["Close"].iloc[-1]),
    "fetch_timestamp": datetime.now(timezone.utc).isoformat(),
    "session":         _detect_session(),
    "news_risk":       False,   # Phase 2 stub
    "news_events":     [],      # Phase 2 stub
}
```
